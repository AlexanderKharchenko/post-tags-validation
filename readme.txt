Plugin checks for the filled post tags.

== Description ==
It does not allow posting posts with an empty tags.
Empty tags is allowed only for drafts and pending.


= Requirements =

* WordPress 3.5.1 or greater
* PHP 5.2.6 or greater (5.3 recommended)
* MySQL 4.1.2 or greater (5.x recommended)

== Installation ==

This plugin uses the standard installation procedure: install the plugin's folder inside of 'wp-content/plugins' (make sure the folder is named *post-tags-validation*).
